Help

• Select Source / 1, 2, 3, 4 keys: Change map source
• Clear button / 0 key: Clear map
• Slider / arrow keys: Select data map date
   D=-5, L=-1, R=+1, U=+5
• Click state: View more poll / projection data

• Edit & Done button / enter key: Edit map
• Party buttons / 0, 1, 2 keys: Select party
• Left click state: Cycle safe, likely, lean, tilt
• Right click state: Cycle tilt, lean, likely, safe
• Hold and drag: Fill states

• Drop image file: Set icon inside pie chart
• R key: Force element resize
